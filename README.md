# SoundWave

A fully responsive, ad-free music streaming web application designed for both desktop and mobile browsers.

## 🚀 Live Demo
🔗 [Launch SoundWave](https://happyreehal.github.io/SoundWave)

## ✨ Features
* **Ad-Free Experience:** Clean and uninterrupted audio player UI without reliance on third-party ads.
* **Custom Playlists:** Implemented personalized playlist creation and song library management.
* **Responsive Design:** Optimized for seamless performance across modern mobile and desktop screens.

## 🛠️ Tech Stack
* **Technologies:** HTML5, CSS3, JavaScript
